М+ - добавить текущее значение на экране к значению в памяти (сложение)
М- -отнять текущее значение на экране к значению в памяти (вычитания)
MRC - вывод значения хранящегося в памяти. Повторное нажатие производит очистку памяти. 

 Обычно в стандартных калькуляторах есть одна ячейка памяти (регистр так сказать), где можно временно хранить одно число для удобства расчетов.

Итак,

MS (Memory Save) - кнопка означает сохранить число, отображенное в данный момент на дисплее калькулятора в память.

MR (Memory Read) - кнопка означает считать число из ячейки памяти и вывести его на дисплей.

MC (Memory Clear) - кнопка означает стереть данные из ячейки памяти.

M+ - прибавить к числу из памяти число, отображенное на дисплее и результат записать в память вместо предыдущего.

M- - вычесть из числа в памяти число, отображенное на дисплее калькулятора и результат записать в память.

4,3,2,1,0: Задает количество десятичных разрядов как 4,3,2,1,0
ADD2: при вводе и отображении результата всех действий сложения и вычитания используется два десятичных разряда. Набрав 2, 
вы получите значение 0,02 во введенных данных. При нажатии кнопки "." во время ввода значения на заданном месте появится десятичная запятая. 
Этот режим не действует для действий умножения и деления.
